No livro PHP: a Bíblia, Tim Converse explica o que é PHP.

O PHP é uma linguagem para a criação de scripts para a Web do lado servidor embutidos em HTLM, Cujo código-font é aberto, e que é compatível com os mais importantes servidores Web (especialmente Apache). O PHP permite incorporar fragmentos de códigos em páginas HTML normais - código esse que é intedrpretado à medida que suas páginas são oferecidas aos usuários. O PHP também serve como uma linguagem que "cola", facilitando a conexão de suas páginas Web com o banco de dados do lado servidor.
